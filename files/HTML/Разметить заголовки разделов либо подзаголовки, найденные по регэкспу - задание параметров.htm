Регэксп для поиска абзацев, которые следует превратить в заголовки либо подзаголовки. Например: «^Глава \d+$».

Предзаданные регэкспы:
учитывать регистр при поиске по регэкспу
сколько захватывать в заголовок либо подзаголовок абзацев, находящихся перед абзацем, найденным по регэкспу
сколько захватывать в заголовок либо подзаголовок абзацев, находящихся после абзаца, найденного по регэкспу
